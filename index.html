自己紹介

はじめまして。私は守谷喜大(@nanjyakore_)と申します。大学生時代に小学生向けのプログラミング教室でiOSアプリ開発の指導を勤めた事がきかっけで、プログラミングの世界に足を踏み入れました。主にiOS向けのネイティブアプリ開発やUnityを使った3Dゲームの開発をしており、言語はSwiftとC#を使用することが多いです。必要であればどんな開発環境にも挑戦していきたいと思っています。

経験したことのある言語

Native: Swift / Objective-C / C# / Java
Web: JavaScript / HTML / CSS
CLI: C / Ruby / Python

ソフトウェアスキル

Adobe: PhotoShop, Illustrator, AfterEffects, PremierePro
Others: MineCraft
